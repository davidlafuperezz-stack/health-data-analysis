# Health Data Analysis Project
## Overview
This project explores a simple health dataset using Python. The goal is to practice data analysis, visualization, and basic insights extraction.
## Dataset
The dataset includes information about:
- Age
- Sex
- Cholesterol levels
- Heart disease presence
## Tools used
- Python
- Pandas
- Matplotlib
- Seaborn
## Analysis performed
- Data exploration (head, info, describe)
- Statistical analysis
- Data visualization (histograms and boxplots)
## Key insight
Higher cholesterol levels are associated with a higher incidence of heart disease in this dataset.
## Note
This is a small educational dataset used for learning purposes.
